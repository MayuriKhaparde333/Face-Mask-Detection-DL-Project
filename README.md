# Face Mask Detection using Deep Learning

📌 Project Overview

This project is a real-time Face Mask Detection system built using Deep Learning and Computer Vision techniques. It detects whether a person is wearing a mask or not using images and live camera feed.

The project is designed to demonstrate practical implementation of AI in real-world safety applications.


🎯 Features

* 📸 Image Upload Detection
* 🎥 Real-Time Camera Detection
* 🧠 Deep Learning Model (CNN)
* 👤 Face Detection using OpenCV
* ⚡ Fast and Optimized Prediction
* 🌐 Web App using Streamlit


 🛠️ Technologies Used

* Python
* TensorFlow / Keras
* OpenCV
* NumPy
* Streamlit
* Streamlit-WebRTC

📂 Project Structure


Face-Mask-Detection/
│
├── app_streamlit.py       # Streamlit Web App
├── detect.py              # Real-time detection (OpenCV)
├── train.py               # Model training
├── mask_detector.h5       # Trained model
├── requirements.txt       # Dependencies
└── dataset/               # Dataset folder



📊 Model Details

* Model Type: Convolutional Neural Network (CNN)
* Input Size: 224x224
* Output Classes:

  * 😷 Mask
  * ❌ No Mask
* Accuracy: ~90%+ (depends on dataset)
